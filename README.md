## 🛠️ Local Development

Clone the repository and install dependencies:

```bash
git clone https://github.com/ChristChad-mv/creative-ai-portfolio.git
cd creative-ai-portfolio
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
